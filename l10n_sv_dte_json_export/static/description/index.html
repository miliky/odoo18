Exportar Adjuntos de Facturas

Descarga masiva de archivos adjuntos en facturas desde Odoo 18

Versión: 18.0.1.0.0

Descripción General

Este módulo permite exportar todos los adjuntos vinculados a facturas desde el formulario de facturas en Odoo. También incluye los archivos JSON FSSE para documentos tipo 14 (Factura de Sujeto Excluido).

Características Principales

Funcionalidad del Asistente

  • Selecciona facturas por fecha o manualmente
  • Exporta todos los adjuntos en un archivo ZIP
  • Incluye documentos XML o JSON como adjuntos

Integración con Facturas

  • Botón directo en el formulario de facturas
  • Compatible con facturas y notas de crédito/débito
  • Compatible con documentos nacionales e internacionales

Requisitos Técnicos

  • Odoo 18.0
  • Módulo base de contabilidad (account)

Uso

Desde el formulario de factura, haz clic en el botón "Descargar adjuntos" ubicado en el encabezado. Luego selecciona el rango de fechas o las facturas específicas, y presiona "Descargar ZIP" para obtener todos los adjuntos en un solo archivo.
