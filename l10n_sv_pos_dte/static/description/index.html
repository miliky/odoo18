Facturación Electrónica para POS - Genere documentos fiscales electrónicos (DTE) automáticamente desde el Punto de Venta, cumpliendo con la normativa de la DGII de El Salvador

Nota importante: Para una instalación limpia, instale primero el módulo El Salvador - Accounting (l10n_sv_cta) antes de instalar este módulo.

Características Principales

  • Generación automática de DTE desde POS
  • Asignación automática del tipo de documento
  • Generación de números de control
  • Creación automática del JSON DTE
  • Soporte para múltiples tipos de documentos
  • Integración con l10n_sv_document_type

Beneficios

  • Cumplimiento fiscal automático desde POS
  • Reducción de errores en la facturación
  • Proceso de facturación más rápido
  • Integración completa con Odoo POS
  • Soporte para múltiples establecimientos
  • Interfaz amigable y fácil de usar

Proceso de Facturación Electrónica en POS

1. Venta en POS

Cree su venta normalmente en el POS de Odoo

2. Facturación

Facture la orden y el sistema asigna el tipo de documento

3. Generación DTE

El sistema genera y envía el JSON DTE automáticamente

4. Confirmación

Reciba el sello de recepción de DGII
